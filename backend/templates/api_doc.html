User Management

Authentication, user profiles, and role management (students, lecturers, admins)

POST /api/users/login/ User login with username/email and password
POST /api/users/logout/ User logout and token cleanup
GET /api/users/profile/ Get current user profile data
PUT /api/users/profile/ Update user profile information
POST /api/users/change-password/ Change user password
GET /api/users/dashboard/ Get user dashboard data by type

Course Management

Course creation, enrollment, topics, and attendance management

GET /api/courses/courses/ List courses for current user
POST /api/courses/courses/ Create new course (lecturers only)
GET /api/courses/courses/{id}/students/ Get enrolled students (lecturers only)
POST /api/courses/course/{id}/upload-students/ Upload CSV to create and enroll students
GET /api/courses/my-courses/ Get user's courses

Traditional Quiz System

Quiz creation, management, live quizzes, and submission

GET /api/quizzes/quizzes/ List quizzes for current user
POST /api/quizzes/quizzes/ Create new quiz (lecturers only)
POST /api/quizzes/quiz/start/ Start quiz attempt (students only)
POST /api/quizzes/quiz/submit/ Submit and finalize quiz attempt
GET /api/quizzes/student/available-quizzes/ Get available quizzes for student

AI-Powered Adaptive Quizzes

Claude AI-generated adaptive learning quizzes from lecture slides

POST /api/ai-quiz/lecturer/upload-slide/ Upload PDF lecture slides
POST /api/ai-quiz/lecturer/generate-questions/ Generate AI questions from slides using Claude API
GET /api/ai-quiz/student/available-slides/ Get available slides with quiz access info
GET /api/ai-quiz/student/quiz/{id}/ Get adaptive quiz questions (progressive difficulty)
POST /api/ai-quiz/student/submit-quiz/ Submit adaptive quiz attempt

Analytics & Engagement

Student engagement tracking, performance analytics, and comprehensive reporting

GET /api/analytics/lecturer/dashboard/ Main lecturer analytics dashboard
POST /api/analytics/lecturer/chart/ Generate specific analytics charts
GET /api/analytics/student/dashboard/ Student personal analytics dashboard
GET /api/analytics/quiz/{id}/stats/ Detailed quiz statistics and question analysis
GET /api/analytics/export/ Export analytics data (CSV/JSON)
GET /api/analytics/trends/engagement/ Engagement trends over time

Student Achievements & Gamification

Badge system, XP tracking, streaks, and student achievement management

GET /api/achievements/dashboard/ Student achievement dashboard with XP, level, and recent badges
GET /api/achievements/badges/ Student's badge collection with filter options (all, earned, gold, legendary)
GET /api/achievements/badges/available/ All available badges and their requirements
GET /api/achievements/history/ Student's achievement history and daily activities
GET /api/achievements/stats/ Detailed achievement statistics and badge breakdowns
POST /api/achievements/check/ Manually trigger achievement checking for student
GET /api/achievements/leaderboard/ Achievement leaderboard by XP, badges, or streak
